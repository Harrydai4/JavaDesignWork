# 通讯录管理系统 — 程序框架说明

> 与源码目录 `src/main/java/com/javadesign/addressbook` 一致，供论文「总体设计 / 系统结构」引用。  
> 更新日期：2026-04-19

---

## 1. 设计目标

- **单机桌面应用**，JDK 25 + Maven；界面 **Swing**。
- **数据持久化**：JSON 文件 + 可选照片文件目录。
- **业务**：联系人、分组（多对多）、搜索（含拼音）、CSV / vCard 导入导出。

---

## 3. 包与主要类职责

| 包 | 类 | 职责 |
|----|-----|------|
| 根 | `AddressBookApp` | 程序入口：设置外观、加载数据、打开 `MainFrame` |
| `model` | `Contact` | 联系人实体，含课程要求全部字段及 `groupIds` |
| `model` | `Group` | 分组实体（id + 名称） |
| `storage` | `AddressBookData` | 持久化根：`contacts` + `groups` 列表 |
| `storage` | `JsonAddressBookStorage` | Jackson 读写 JSON；默认目录 `%USERPROFILE%\.javadesign-addressbook\` |
| `service` | `AddressBookService` | 增删改查、分组、导入合并、保存加载 |
| `service` | `ContactSearch` | 过滤、排序；`matches` 供测试；`surnameBucketKey` 供「按姓分类」扩展 |
| `util` | `PinyinUtil` | 拼音全拼、首字母、电话数字抽取 |
| `importexport` | `CsvContactCodec` | CSV 转义与解析 |
| `importexport` | `VCardContactCodec` | vCard 块解析与生成 |
| `ui` | `MainFrame` | 左树右表、菜单、工具栏、导入导出、列显示 |
| `ui` | `ContactEditDialog` | 表单编辑、照片复制到 `photos/` |

---

## 4. 运行时数据流

1. **启动**：`AddressBookApp.main` → `JsonAddressBookStorage.load()` → `AddressBookService` 持有内存中的 `AddressBookData`。
2. **操作**：用户在 `MainFrame` 操作 → 调用 `AddressBookService` → 修改内存列表 → 关键操作后 `save()` 写回 JSON。
3. **退出**：窗口关闭事件内 `save()`，再 `System.exit(0)`。

**检查点（持久化）**：若 `save()` 抛 `IOException`，界面以 `JOptionPane` 中文提示。

## 7. 与课程功能对应（简表）

| 课程要求 | 实现位置 |
|----------|----------|
| 左侧分组 + 右侧列表 | `MainFrame` |
| 联系人增删改 | `MainFrame` + `ContactEditDialog` + `AddressBookService` |
| 分组增删（删组不删人） | `AddressBookService.removeGroup` |
| 多选入组 / 出组 | `MainFrame` + `AddressBookService` |
| 搜索（含拼音） | `ContactSearch` + `PinyinUtil` |
| CSV / vCard 导入导出 | `CsvContactCodec` / `VCardContactCodec` + 菜单 |
| 列显示切换 | `MainFrame` 视图菜单 |

**可扩展**：题目要求「按姓/拼音首字母**分类**展示」——当前表格为**排序**；可在 UI 层按 `ContactSearch.surnameBucketKey` 分块渲染。
